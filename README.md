# 🥛 KTS Milk - Pure Buffalo Milk Delivery Website

A modern responsive website for a milk delivery business.  
Built using HTML, Tailwind CSS, and JavaScript with WhatsApp ordering system.

## ✨ Features

- 📱 Fully responsive (mobile + desktop)
- 🥛 Modern Flipkart-style UI design
- 💚 WhatsApp order integration
- 🚚 Daily milk delivery sections
- 📦 Product cards (1L, 500ml, Curd)
- ⭐ Customer testimonials
- 📞 Contact section (phone, WhatsApp, location)
- 🎨 Smooth animations and clean UI

---

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
- JavaScript
- Lucide Icons
- Google Fonts (Outfit)

```bash

```

3. Open `index.html` in browser
